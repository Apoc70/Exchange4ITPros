# Exchange Online

Scripts for Exchange Online, designed to run through the Exchange Online PowerShell module (`ExchangeOnlineManagement`).

## Categories

| Category | Description |
| --- | --- |
| Reporting | Scripts that collect and report on Exchange Online configuration and usage data |
| Compliance | Scripts that support compliance, retention, and auditing scenarios |
| Automation | Scripts that automate recurring Exchange Online administration tasks |

Each category folder contains a short overview README with links to the individual scripts. Each script has its own folder with the script file and a dedicated README covering purpose, parameters, examples, and prerequisites.

## Requirements

Scripts in this folder generally require the `ExchangeOnlineManagement` PowerShell module and a connection established through `Connect-ExchangeOnline`. Specific module or version requirements, if any, are documented in each script's header and README.
