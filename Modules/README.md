# Modules

Reusable PowerShell modules and shared helper content used by scripts in this repository.

## Modules

| Module | Description |
| --- | --- |
| [GlobalFunctions](GlobalFunctions/) | Shared PowerShell functions used by other scripts and tools |

## Requirements

Module requirements vary by project. Check the README and comment-based help in each module or script for installation and usage details. If you keep the folder in a custom location, use [GlobalFunctions/Set-PersistentPSModulePath.ps1](GlobalFunctions/Set-PersistentPSModulePath.ps1) to add the module path persistently on supported Windows systems.
