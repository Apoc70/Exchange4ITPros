# Entra

Scripts for Entra ID, mostly designed to run through the Microsoft Graph PowerShell SDK.

## Categories

| Category | Description |
| --- | --- |
| Reporting | Scripts that collect and report on Entra ID configuration and objects |
| AppRegistrations | Scripts covering app registrations, enterprise applications, and related permissions |
| Automation | Scripts that automate recurring Entra ID administration tasks |

Each category folder contains a short overview README with links to the individual scripts. Each script has its own folder with the script file and a dedicated README covering purpose, parameters, examples, and prerequisites.

## Requirements

Scripts in this folder generally require the Microsoft Graph PowerShell SDK and appropriate Graph permissions for the scopes used. Specific module, scope, or version requirements are documented in each script's header and README.
