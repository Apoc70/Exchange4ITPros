# SharePoint Online

Scripts for SharePoint Online, designed to run through PnP PowerShell.

## Categories

| Category | Description |
| --- | --- |
| PnP | Scripts using PnP PowerShell for SharePoint Online administration and automation |

Each category folder contains a short overview README with links to the individual scripts. Each script has its own folder with the script file and a dedicated README covering purpose, parameters, examples, and prerequisites.

## Requirements

Scripts in this folder generally require the `PnP.PowerShell` module and a connection established through `Connect-PnPOnline`. Specific module or version requirements, if any, are documented in each script's header and README.
